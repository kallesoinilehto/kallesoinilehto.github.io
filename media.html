I have a lot of experience from 3D modeling. I've worked with both Blender and Rhino3D, modeling 3D prints and video game assets. The tablet you see here is the latest of my 3D models. During the project I learned to create UV maps by UV unwrapping objects. That way it was easy to move the materials from Blender to Unity. My modeling process has been heavily reliant on mesh primitives like cubes and cylinders. After creating a model with primitives, I use tools like bevel for edges and modifiers like boolean for editing objects.
